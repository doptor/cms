Welcome to RyaanCMS

RyaanCMS is the world's first free, open-source AI-powered Laravel CMS that runs on shared hosting. Build complex web applications — e-commerce stores, SaaS platforms, blogs, portfolios — just by typing a prompt.

✅ No VPS required. RyaanCMS runs on any cPanel shared hosting with PHP 8.2+ and MySQL — which you likely already have.

Core Architecture

  • Backend: Laravel 11 · PHP 8.2+
  • Database: MySQL (free with cPanel)
  • AI: Claude Sonnet 4 + DeepSeek V3 (BYOK)
  • Frontend: Tailwind CSS CDN + Alpine.js (no npm build)
  • Deploy: FTP auto-deploy to shared hosting
{{-- INSTALLATION --}}

Installation

Requirements

  • PHP 8.2 or higher
  • MySQL 5.7+ / MariaDB 10.3+
  • Composer 2.x
  • PHP extensions: pdo_mysql, openssl, ftp, mbstring, curl

One-Line CLI Install

curl -fsSL https://ryaancms.com/install | php

Manual Install

# 1. Clone the repository
git clone https://github.com/ryaancms/ryaancms.git
cd ryaancms

# 2. Install PHP dependencies
composer install --optimize-autoloader --no-dev

# 3. Configure environment
cp .env.example .env
php artisan key:generate

# 4. Edit .env with your database credentials
nano .env

# 5. Run database migrations
php artisan migrate --seed

# 6. Set permissions & create storage link
chmod -R 755 storage bootstrap/cache
php artisan storage:link

# 7. Start development server
php artisan serve
ℹ️ Tip: Visit http://localhost:8000 then go to Settings → API Keys to add your Claude or DeepSeek key before building your first project.
{{-- SHARED HOSTING --}}

Shared Hosting Setup

RyaanCMS is designed to run on any cPanel shared hosting. This guide covers Hostinger, Bluehost, Namecheap, and similar providers.

Step 1 — Upload Files

Upload all files except the public/ folder to a directory like laravel/ in your home directory. Upload the contents of public/ directly to public_html/.

Step 2 — Fix the index.php paths

// public_html/index.php — update both paths:
require __DIR__.'/../laravel/vendor/autoload.php';
$app = require_once __DIR__.'/../laravel/bootstrap/app.php';

Step 3 — Database

In cPanel → MySQL Databases: create a database, create a user, assign ALL privileges, then update your .env:

DB_HOST=127.0.0.1
DB_DATABASE=username_dbname
DB_USERNAME=username_user
DB_PASSWORD=yourpassword

Step 4 — Run Migrations

Use cPanel Terminal (if available) or a web-based Artisan runner:

php artisan migrate --force --seed
⚠️ PHP Version: Make sure PHP 8.2+ is selected in cPanel → MultiPHP Manager for your domain.
{{-- AI API KEYS --}}

AI API Keys Setup

RyaanCMS supports Claude (Anthropic) and DeepSeek. You bring your own API keys — they are stored encrypted with AES-256 and never leave your server.

Getting a Claude API Key

  • Go to console.anthropic.com
  • Create an account and navigate to API Keys
  • Click Create Key and copy it (starts with sk-ant-api03-)
  • Paste it in RyaanCMS → Settings → API Keys

Getting a DeepSeek API Key

  • Go to platform.deepseek.com
  • Register — you get 5 million free tokens on signup
  • Go to API Keys → Create new key
  • Paste it in RyaanCMS → Settings → API Keys

Smart Routing (Auto Mode)

When set to Auto, RyaanCMS routes requests intelligently:

  • DeepSeek V3 — Simple tasks, short prompts, content generation (~80% of requests)
  • Claude Sonnet — Complex code, security, architecture, long prompts (~20%)

This reduces your AI costs by up to 80% compared to using Claude for everything.

{{-- WHITE LABEL --}}

White-Label Mode

Agencies can rebrand RyaanCMS with their own name, logo, colors, and custom domain — completely removing all RyaanCMS branding.

Enable White-Label

# .env
WHITELABEL_ENABLED=true
WHITELABEL_NAME="AgencyCMS"
WHITELABEL_TAGLINE="Your Agency's CMS Platform"
WHITELABEL_LOGO="https://youragency.com/logo.png"
WHITELABEL_COLOR="#e74c3c"
WHITELABEL_COLOR2="#c0392b"
WHITELABEL_HIDE=true

Multi-tenant / Per-Domain

Store white-label configs per domain in the whitelabel_configs database table:

INSERT INTO whitelabel_configs (domain, config, is_active) VALUES (
  'cms.clientsite.com',
  '{"name":"ClientCMS","color":"#3498db","logo":"https://...","hide_branding":true}',
  1
);
{{-- CLI --}}

CLI Installer

RyaanCMS ships with a PHP CLI installer that makes setup on shared hosting simple.

Available Commands

php ryaan install          # Full interactive installation
php ryaan install --quick  # Install with default settings
php ryaan check            # Check system requirements
php ryaan update           # Update to latest version
php ryaan key:generate     # Regenerate APP_KEY
