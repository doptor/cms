# Security Policy

## Reporting a Vulnerability

**Please do NOT report security vulnerabilities through public GitHub issues.**

Email: security@ryaancms.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We'll respond within 48 hours and issue a fix within 7 days for critical issues.

## Scope

- API key encryption bypass
- Authentication bypass
- SQL injection
- XSS vulnerabilities
- CSRF bypass
- FTP credential exposure

## Out of Scope

- Bugs in user-installed marketplace items
- Issues requiring physical access to the server
- Social engineering attacks
